# Study-Central-website
 StudyCentral is a web-based academic resource platform developed to centralize study materials for Computer Science students. The website provides organized access to course syllabi, subject-wise notes, and previous year question papers (PYQs) through a clean and responsive interface.

 
Structured Navigation: Consistent header and footer across all pages for seamless browsing.

Course Repository: Dedicated sections for each subject including Data Structures, Database Systems, Software Engineering, and more.

Syllabus Access: Linked PDF syllabi for all core subjects.

Resource Hub: Downloadable PYQs categorized by subject for efficient exam preparation.

Modern UI/UX: Minimalist design, responsive layout, and grid-based organization.
